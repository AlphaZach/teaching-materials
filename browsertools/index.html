JS202 JS DOM Browser Tools



Overview

This class is a hands-on and follow-along class, so be prepared to follow along with your browser tools. This class is designed around Google Chrome, but the tools we learn today can easily be applied to other browsers. For ease of following along however, please use Google Chrome.

  1. Start by opening your Chrome Developer Tools by doing any of the following
    • Select the Chrome menu Chrome Menu at the top-right of your browser window, then select Tools > Developer Tools.
    • Right-click on any page element and select Inspect Element.
    • Pressing Alt + Cmd + i
  2. On your worksheet, take note of the different panes we'll be covering. Use the elements pane to add those names to the HTML version of the worksheet

The Network Tab

  1. Open coursera.org and open your Dev Tools. Navigate to the network tab.
  2. Click on the XHR filter, and refresh the page.
  3. Look at the headers, preview, response, cookies, and timing panes. The preview pane should provide an interactive json object you can look through for debugging.
  4. Take a look at the images filter as well, and note that not every image as a request issued for it if you have been to the page before.
